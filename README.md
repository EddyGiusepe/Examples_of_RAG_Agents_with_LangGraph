# <h1 align="center"><font color="red">Examples of RAG Agents with LangGraph</font></h1>

<font color="pink">Senior Data Scientist.: Dr. Eddy Giusepe Chirinos Isidro</font>

RAG Agents com LangGraph são sistemas avançados que combinam a técnica de Recuperação-Geração Aumentada (RAG) com a orquestração baseada em grafos proporcionada pelo LangGraph. O LangGraph é uma biblioteca Python que permite criar agentes de IA com lógica complexa, modelando seus comportamentos como grafos formados por nós (ações ou operações) e arestas (fluxos de decisão), com estado compartilhado e persistente.
